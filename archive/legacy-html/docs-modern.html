ACT Placemat Platform

Comprehensive project management and impact storytelling platform

ACT Placemat is a modern web application designed to manage community-focused projects, track their impact through storytelling, and visualize opportunities across Australia. The platform integrates with Notion for project management and Airtable for impact story collection.

Key Features

  • Project Management: Real-time project tracking with revenue and impact metrics
  • Impact Stories: Community story collection and visualization
  • Opportunity Tracking: Grant and partnership opportunity management
  • Geographic Visualization: Interactive maps showing project distribution
  • Analytics Dashboard: Comprehensive performance metrics and insights

System Status

Live Production system running on Node.js with Express
Live Notion integration for project data
Live Airtable integration for stories
Beta Advanced analytics and reporting
Planned Mobile application

Technology Stack

  • Frontend: Modern HTML5, CSS3, Vanilla JavaScript
  • Backend: Node.js with Express framework
  • Database: Notion (Projects), Airtable (Stories)
  • Styling: Custom CSS with Apple-inspired design system
  • APIs: RESTful API endpoints with JSON responses

Quick Start Guide

Prerequisites

  • Node.js 16+ installed
  • Notion account with API access
  • Airtable account with base access

Installation

npm install cp .env.example .env # Configure your API keys in .env npm start

Environment Variables

Variable Description Required
NOTION_TOKEN Notion integration token Yes
NOTION_DATABASE_ID Projects database ID Yes
AIRTABLE_API_KEY Airtable API key Yes
AIRTABLE_BASE_ID Stories base ID Yes
PORT Server port (default: 4000) No

API Reference

The ACT Placemat API provides RESTful endpoints for accessing project data, stories, and system health information.

Base URL

http://localhost:4000/api

Authentication

Currently, the API uses server-side authentication with Notion and Airtable. No client authentication is required for read operations.

Response Format

All API responses are in JSON format with the following structure:

{ "projects": [...], "summary": { "totalProjects": 49, "activeProjects": 23, "totalRevenue": 560300, "totalPotential": 1651500 }, "timestamp": "2025-01-17T12:00:00.000Z" }

Common Endpoints

GET /api/health
System health check and configuration status
GET /api/notion/real-projects
Retrieve all projects from Notion database
POST /api/notion/query
Query Notion database with filters and sorting

Notion Integration

The platform integrates with Notion to manage project data, providing real-time synchronization and rich project information.

Database Schema

The Notion projects database includes the following properties:

Property Type Description
Name Title Project name
Status Select Active, Ideation, Planning, Completed
Revenue Actual Number Actual revenue received
Revenue Potential Number Potential revenue opportunity
State Select Australian state/territory
Place Select Project area classification

Setup Instructions

  1. Create a Notion integration at notion.so/my-integrations
  2. Copy the integration token to your .env file
  3. Share your projects database with the integration
  4. Copy the database ID from the URL to your .env file

Airtable Stories Integration

Airtable integration enables community impact story collection and visualization connected to projects.

Base Structure

  • Stories Table: Individual impact stories with metadata
  • Storytellers Table: Community members contributing stories
  • Project Links: Connections between stories and projects

API Integration

// Example story retrieval const stories = await airtableConnector.getStoriesForProject(projectName); console.log(`Found ${stories.length} stories for ${projectName}`);

System Architecture

ACT Placemat follows a modern, modular architecture designed for scalability and maintainability.

Component Overview

  • Frontend: Static HTML/CSS/JS with modern design system
  • API Layer: Express.js server with RESTful endpoints
  • Data Layer: Notion (projects) + Airtable (stories)
  • Presentation: Real-time dashboard with interactive visualizations
