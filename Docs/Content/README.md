## üìù Content Guidelines

### **ACT Content Standards**
Based on core values, all content should demonstrate:

1. **Radical Humility** - Community voices lead, authentic tone
2. **Decentralised Power** - Show how tools can be owned by communities  
3. **Creativity as Disruption** - Innovative approaches to storytelling
4. **Uncomfortable Truth-Telling** - Honest, unfiltered impact stories

### **Content Principles**
- **No marketing language** - Authentic community voices only
- **Story-first approach** - Begin with community experiences
- **Transparent metrics** - Real data, honest about challenges
- **Replication focus** - How others can adapt and use approaches
