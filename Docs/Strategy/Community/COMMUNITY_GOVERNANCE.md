## ü§ù Decision Making Process

### Consent-Based Decisions
Not everyone has to agree, but no one strongly objects.

### Decision Levels

#### Level 1: Individual (Just do it)
- Using the platform
- Sharing resources
- Helping others
- Personal learning
- Local innovations

#### Level 2: Working Group (Propose and proceed)
- New features
- Resource creation
- Process improvements
- Pilot projects
- Experiments

#### Level 3: Regional Circle (Consult and proceed)
- Regional strategies
- Local partnerships
- Resource allocation
- Training programs
- Regional innovations

#### Level 4: Community Assembly (Full consent process)
- Governance changes
- Major platform changes
- Core value updates
- Open source timeline
- International expansion

### Consent Process
1. **Proposal** - Someone proposes an idea
2. **Questions** - Clarifying questions only
3. **Reactions** - Initial thoughts shared
4. **Amend** - Proposal modified based on input
5. **Consent Round** - Each person asked for consent
6. **Integration** - Address any objections
7. **Celebration** - Acknowledge decision made

### Objection Criteria
Valid objections:
- Harms the community
- Violates core values
- Creates inequality
- Reduces accessibility
- Blocks others' participation

Not valid objections:
- Personal preference
- "Not perfect"
- "I have a better idea"
- Fear of change
- Protecting territory

## üí∞ Sustainability & Resources

### Financial Sustainability
**Sliding Scale Contributions:**
- Large orgs: $500-2000/month
- Medium orgs: $200-500/month
- Small orgs: $50-200/month
- Individuals: $0-50/month
- Those in need: $0 + other contributions

**Alternative Contributions:**
- Training others (valued at $200/session)
- Creating resources (valued at $100/resource)

- Technical support (valued at $150/hour)
- Story sharing (valued at $50/story)

### Resource Distribution
- 30% - Platform maintenance
- 30% - Regional support
- 20% - Training programs
- 10% - Innovation fund
- 10% - Emergency support

### Transparency
- Monthly financial reports
- All spending visible
- Community approval for large expenses
- Annual audit by community member
- No hidden anything
