## **Progress Tracking Conventions**

### Status Indicators
- [ ] **Todo** - Not started
- [-] **In Progress** - Being worked on (with timestamp)
- [x] **Completed** - Done (with completion timestamp)
- [!] **Blocked** - Cannot proceed (with blocker description)
- [~] **Deferred** - Moved to later phase

### Session Management
- **Current session focus:** One primary task only
- **Session length:** Max 20 minutes before context review
- **Handoff protocol:** Update this roadmap at session end
- **Context preservation:** Key decisions recorded here

### Priority Framework (RICE)
- **Reach:** How many users/processes affected (1-100)
- **Impact:** Business goal achievement (1-5)  
- **Confidence:** Execution confidence (100%, 80%, 50%)
- **Effort:** Implementation complexity (1-10)
- **Score:** (Reach Ã— Impact Ã— Confidence) / Effort
