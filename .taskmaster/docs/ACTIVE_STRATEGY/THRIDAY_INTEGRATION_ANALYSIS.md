## FAQ: Thriday for A Curious Tractor

### Q: Can Thriday handle our R&D tax incentive claims?
**A**: No, not directly. Thriday has no specific R&D tax features. You'll need:
- Manual R&D expense tracking outside Thriday
- Separate R&D documentation system
- R&D tax specialist for annual claims ($5,000-8,000)

### Q: Does Thriday support cooperatives and multiple SPVs?
**A**: Unclear. Thriday mentions "multiple business entities under one login" but doesn't specify if this means separate legal entities or just departments. For a cooperative + multiple SPVs, you may need:
- Separate Thriday account per entity ($29.95 each), OR
- Use Xero for complex multi-entity structure

### Q: Can we integrate Thriday with our AI assistant?
**A**: Not easily. Thriday has no public API. Options:
- Manual CSV exports (monthly reconciliation)
- Request API access from Thriday (may not exist)
- Use Xero instead for full API access

### Q: What about community transparency dashboards?
**A**: Thriday doesn't support public dashboards. You'll need to:
- Export data manually from Thriday
- Build custom dashboard with AI assistant
- Update monthly (not real-time)

### Q: Is Thriday worth it despite the limitations?
**A**: It depends:
- **Yes if**: You prioritize simplicity, cost savings, and can accept manual R&D tracking
- **No if**: You need full automation, R&D compliance, and API integrations
- **Hybrid if**: You want best of both worlds (Thriday for daily ops, Xero for compliance)
