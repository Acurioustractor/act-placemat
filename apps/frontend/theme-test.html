Code Example

Here's some inline code styling and a code block:

function toggleTheme() {
  const body = document.body;
  const isPublic = body.getAttribute('data-theme') === 'public';
  body.setAttribute('data-theme', isPublic ? 'internal' : 'public');
}

Theme Usage

Public Theme (Elegant & Editorial)

  • Inspired by Oliver Jeffers and Baseborn Studio
  • Uses Playfair Display for headings
  • Generous spacing and elegant shadows
  • Clean, borderless components
  • Perfect for public-facing content

Internal Theme (Practical & Efficient)

  • Inspired by Linear, GitHub, and Notion
  • Uses Inter throughout for consistency
  • Compact spacing and clear borders
  • Structured, functional components
  • Perfect for admin interfaces and tools
