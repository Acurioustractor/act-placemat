# ACT Project Portfolio - Webflow Cloud App

This is a Next.js app built for Webflow Cloud integration.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001)

## Environment Variables

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:4000
```

## Deploy to Webflow Cloud

This app is designed to be deployed via Webflow Cloud.

1. Push to GitHub
2. Webflow Cloud auto-deploys from `apps/webflow-portfolio` directory
3. Available as a component in Webflow Designer

## Features

- 72+ projects from Notion
- Filter by theme and status
- Animated project cards
- Responsive design
- Real-time data sync
